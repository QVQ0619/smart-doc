# 聊天前端居中规则文件入库（P-B）Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在右栏聊天 ChatView 上方加一条「规则文件入库」操作条：选文件→传 agent sandbox 并发消息让 AI 识别/建议→用户点确认→前端直接调本地后端入库。

**Architecture:** P-B 前端居中。新增 `RuleDocChatBridge` 组件，用 SDK 的 `uploadFiles`/`useChat().send`/`buildMessageContent` 做 agent 侧（best-effort），用已有 `uploadStandardDocs` 做本地入库（关键路径，与 agent 侧解耦）。后端零改动、Blade 侧零配置。

**Tech Stack:** React 19 + TS + Ant Design 5 + sonner + `@blade-hq/agent-kit@0.5.11`（`/react` 导出 `uploadFiles`/`useChat`/`buildMessageContent`）+ vitest。

## Global Constraints

- **后端零改动、Blade 侧零配置**：唯一本地依赖是已存在的 `uploadStandardDocs(files: File[]) -> Promise<UploadResult>`（`UploadResult = { uploaded: StandardDoc[]; failed: UploadFailed[] }`，`StandardDoc` 含 `doc_code`/`title`/`file_name`）。
- SDK 从 `@blade-hq/agent-kit/react` 导入：`uploadFiles(sessionId, dirPath, files: File[])`（Promise）、`useChat(sessionId) -> { send }`、`buildMessageContent(text) -> MessageContent`；`send` 是同步 `void`。
- sandbox 目录常量 = `"uploads"`；选文件后**自动发**提示消息。
- **解耦原则**：本地入库只依赖前端手里的 `File`，`uploadFiles`/`send` 失败不得阻断「存入规则库」。
- 决策门：仅用户点「存入规则库」才入库，绝不自动入库。
- 通知用 `sonner` 的 `toast`（与 `StandardDocLibrary` 一致）。
- 前端命令 CWD = 仓库根；测试套件较慢（全量约 1~1.5 分钟），命令给足超时。
- 现有前端 27 测试不回归；后端不动。
- 不要 `git add -A`：仓库有未跟踪的 `docs/file/`、`docs/参考文件/`，逐文件 `git add`。

- [ ] **Step 3: 写实现 `src/components/RuleDocChatBridge.tsx`**

```tsx
import { useRef, useState, type ChangeEvent } from "react";
import { Button } from "antd";
import { UploadOutlined, SaveOutlined } from "@ant-design/icons";
import { toast } from "sonner";
import { uploadFiles, useChat, buildMessageContent } from "@blade-hq/agent-kit/react";
import { uploadStandardDocs } from "../api/standardDocs";

const SANDBOX_DIR = "uploads";

export default function RuleDocChatBridge({ sessionId }: { sessionId: string }) {
  const inputRef = useRef<HTMLInputElement>(null);
  const [pickedFile, setPickedFile] = useState<File | null>(null);
  const [aiBusy, setAiBusy] = useState(false);
  const [saving, setSaving] = useState(false);
  const { send } = useChat(sessionId);

  async function onPick(e: ChangeEvent<HTMLInputElement>) {
    const file = e.target.files?.[0];
    e.target.value = "";
    if (!file) return;
    setPickedFile(file);
    setAiBusy(true);
    try {
      await uploadFiles(sessionId, SANDBOX_DIR, [file]);
      send(
        buildMessageContent(
          `我上传了 ${file.name}，请判断它是不是规则文档（政策/指南/规范），并建议是否存入规则库。`,
        ),
      );
    } catch {
      toast.warning("传给 AI 失败，但你仍可直接存入规则库");
    } finally {
      setAiBusy(false);
    }
  }

  async function onSave() {
    if (!pickedFile) return;
    setSaving(true);
    try {
      const res = await uploadStandardDocs([pickedFile]);
      if (res.uploaded.length) {
        const d = res.uploaded[0];
        toast.success(`已存入规则库：${d.title}（${d.doc_code}）`);
        try {
          send(buildMessageContent(`已存入规则库：${d.title}（${d.doc_code}）`));
        } catch {
          /* 反映到对话失败不影响入库结果 */
        }
      }
      if (res.failed.length) {
        toast.warning(
          `${res.failed.length} 个失败：` +
            res.failed.map((f) => `${f.name}(${f.reason})`).join("，"),
        );
      }
      setPickedFile(null);
    } catch (e) {
      toast.error(e instanceof Error ? e.message : String(e));
    } finally {
      setSaving(false);
    }
  }

  return (
    <div
      style={{
        display: "flex",
        alignItems: "center",
        gap: 8,
        padding: "8px 16px",
        borderBottom: "1px solid #e5e6eb",
        flex: "0 0 auto",
      }}
    >
      <input ref={inputRef} type="file" hidden onChange={onPick} />
      <Button
        size="small"
        icon={<UploadOutlined />}
        loading={aiBusy}
        onClick={() => inputRef.current?.click()}
      >
        选规则文件给 AI 看
      </Button>
      <Button
        size="small"
        type="primary"
        icon={<SaveOutlined />}
        disabled={!pickedFile || saving}
        loading={saving}
        onClick={() => void onSave()}
      >
        存入规则库
      </Button>
      <span style={{ color: "#86909c", fontSize: 12 }}>
        {pickedFile ? `已选：${pickedFile.name}` : "未选文件"}
      </span>
    </div>
  );
}
```
